## Theme

Our theming system is designed to facilitate automatic theme transitions, leveraging CSS variables to adapt to user preferences or system settings seamlessly. This dynamic adaptation ensures a uniform and accessible user experience across both light and dark modes, eliminating the need for manual adjustments.

Below, we detail the methodology our design system employs to define these variables. Modifying the colors within the `Theme Context` initiates an automatic update of the theme colors listed below to reflect your customizations.

```
body.dark {
  --primary-background-color: var(--dark-shades-color);
  --secondary-background-color: var(--dark-accent-color);
  --primary-text-color: var(--light-shades-color);
  --secondary-text-color: var(--light-accent-color);
  --link-color: var(--brand-accent-color);
  --link-hover-color: var(--brand-color);
}

body.light {
--primary-background-color: var(--light-shades-color);
--secondary-background-color: var(--light-accent-color);
--primary-text-color: var(--dark-shades-color);
--secondary-text-color: var(--dark-accent-color);
--link-color: var(--brand-color);
--link-hover-color: var(--brand-accent-color);
}
```
