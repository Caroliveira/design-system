## Theme

Our theming system is designed to automatically switch themes based on the CSS variables used. By incorporating the variables listed below, themes will adjust dynamically to match user preferences or system settings. This seamless transition ensures a consistent and accessible user experience across light and dark modes without manual intervention.

```
body.dark {
  --primary-background-color: var(--dark-shades-color);
  --secondary-background-color: var(--dark-accent-color);
  --primary-text-color: var(--light-shades-color);
  --secondary-text-color: var(--light-accent-color);
}

body.light {
  --primary-background-color: var(--light-shades-color);
  --secondary-background-color: var(--light-accent-color);
  --primary-text-color: var(--dark-shades-color);
  --secondary-text-color: var(--dark-accent-color);
}
```
